
# lionshare-api

A fork of lionsharecapital/lionshare-api

## Getting Started

### Requirements

mandatory

- `Redis` must me installed and running.

recommended

- install `pm2` globally `npm install -g pm2`


### Installation

```shell
git clone https://github.com/dearfrankg/lionshare-api.git
cd lionshare-api
yarn
npm start
```
